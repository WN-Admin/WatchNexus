## Installation

```bash
pip install wn-garnish
```

## Dependencies

- wn-core>=1.0.0
- beautifulsoup4>=4.12.0

## API Endpoints

- `GET /api/garnish/search/tv`
- `GET /api/garnish/search/movie`
- `POST /api/garnish/download`
