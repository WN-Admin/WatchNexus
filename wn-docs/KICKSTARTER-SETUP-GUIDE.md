## 1. Account Setup

### Before You Start
1. Go to [kickstarter.com](https://www.kickstarter.com)
2. Click **Start a project**
3. Verify your identity (government ID required)
4. Connect bank account for payouts
5. Verify email address

### Creator Profile
- **Profile Photo**: Use professional headshot or WatchNexus logo
- **Bio**: "Creator of WatchNexus - the unified media server"
- **Location**: Your city/country
- **Website**: Link to demo or landing page

## 8. FAQ

### Page: FAQ

Add these questions:

---

**Q: Is WatchNexus really free/open source?**
```
A: Yes! WatchNexus is MIT licensed and always will be. The Kickstarter funds development of mobile apps and new features. You can self-host the web app for free today.
```

---

**Q: What about legal issues with downloading content?**
```
A: WatchNexus is a media organization and playback tool. It does not host any content. Users are responsible for ensuring they have the legal right to access content. We do not promote piracy.
```

---

**Q: Do I need technical skills to use WatchNexus?**
```
A: Basic computer skills are needed for initial setup (similar to installing any software). Once running, the interface is designed to be user-friendly for everyone.
```

---

**Q: Can I use my existing Sonarr/Radarr setup?**
```
A: WatchNexus is a replacement, not an integration. However, you can run them side-by-side during transition. Import tools are planned for future updates.
```

---

**Q: What about Plex/Jellyfin clients?**
```
A: WatchNexus includes a Jellyfin-compatible API layer! Your existing Jellyfin clients can connect to WatchNexus.
```

---

**Q: When will mobile apps be available?**
```
A: Android apps are targeted for 2-3 months after campaign ends, assuming we hit the stretch goal. We'll provide monthly progress updates.
```

---

**Q: Is there a subscription?**
```
A: No subscriptions, ever. WatchNexus is self-hosted software you run on your own hardware. One-time backing gets you lifetime access.
```

---

**Q: What if the project fails?**
```
A: The core WatchNexus application already exists and works. Kickstarter funds expand it. Even without funding, the current version will remain available on GitHub.
```
