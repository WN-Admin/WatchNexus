## Library Management

### Adding Media Libraries

1. Go to **Settings** ‚Üí **Library** tab
2. Click **Add Library**
3. Enter:
   - Library name (e.g., "Movies", "Anime")
   - Media type (Movies, TV, Anime, Music)
   - Folder path (e.g., `/media/movies`)
4. Click **Add Library**
5. Click **Scan** to index content

### Supported Media Types

| Type | Extensions | Naming Convention |
|------|------------|-------------------|
| Movies | .mp4, .mkv, .avi, .mov | `Movie Name (Year).ext` |
| TV Shows | .mp4, .mkv, .avi | `Show Name/Season 01/S01E01.ext` |
| Anime | .mp4, .mkv | `Anime Name/Season 01/Episode 01.ext` |
| Music | .mp3, .flac, .m4a | `Artist/Album/Track.ext` |
| Audiobooks | .mp3, .m4b | `Author/Book Name/Chapter.ext` |

### Library Sub-tabs

| Tab | Purpose |
|-----|---------|
| **Libraries** | Manage folders and scanning |
| **Media Management** | File organization settings |
| **Quality Profiles** | Download quality preferences |
| **Mass Editor** | Bulk edit multiple items |
| **Manual Import** | Import files from downloads |

## Searching & Downloading

### Using Discover

1. Go to **Discover** in sidebar
2. Search by title, actor, or keywords
3. Filter by:
   - Media type (Movie/TV)
   - Year range
   - Genre
   - Rating

### Adding to Library

1. Find content in Discover
2. Click **Add to Library** (+ icon)
3. Choose quality profile
4. Select destination library
5. Content will be searched and downloaded

### Download Management

View downloads at **Downloads** page:

## User Profiles

### Creating Profiles

1. Go to **Settings** ‚Üí **Users**
2. Click **Add Profile**
3. Enter name and select avatar
4. Set permissions if needed

### Profile Features

Each profile has:
- Separate watchlist
- Individual watch history
- Personal "Continue Watching"
- Custom preferences

### Switching Profiles

- Click your profile icon (top right)
- Select another profile
- Or use the profile selector on local login

## Tips & Tricks

### Faster Navigation

- Use keyboard shortcuts for quick access
- Pin favorite libraries to sidebar
- Set your preferred start page

### Better Recommendations

- Rate movies/shows you've watched
- Keep your watchlist updated
- Watch history improves suggestions

### Download Quality

- Set up quality profiles for different types
- Use "Preferred" and "Acceptable" quality tiers
- Enable "Upgrade Until" for better versions

### Subtitle Management

- Configure multiple subtitle providers
- Set preferred languages in order
- Enable auto-download for new content

### Remote Access

- Use Gelatin for secure remote access
- Set up HTTPS for encrypted connections
- Consider a VPN for additional privacy
