React-based frontend for WatchNexus media server.

## Tech Stack

- React 18
- TailwindCSS
- Shadcn/UI
- Framer Motion
- React Router

## Installation

```bash
# Install dependencies
yarn install

# Start development server
yarn start

# Build for production
yarn build
```

## Configuration

Copy `.env.example` to `.env` and configure:

```bash
REACT_APP_BACKEND_URL=http://your-server:8001/api
```
