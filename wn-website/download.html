Quick Start Guide

1. Download and Install

Download the appropriate version for your platform. Run the installer or extract the archive.

2. Start WatchNexus

Launch WatchNexus from your applications menu or run the startup script. The web interface will be available at http://localhost:8001

3. Configure Your Libraries

Go to Settings ‚Üí Library to add your media folders. WatchNexus will scan and organize your existing content.

4. Add Indexers

Go to Settings ‚Üí Indexers to add your preferred torrent or usenet indexers. Test connections to ensure everything works.
